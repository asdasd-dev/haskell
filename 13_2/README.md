Описать функцию renumerate, которая в данном списке целых чисел заменяет все числа на порядковые номера 1,2,3 и т.д.

Пример вызова:
renumerate (Node 35 (Node 12 Empty Empty) (Node 70 Empty Empty)) 
Результат должен быть равен Node 1 (Node 2 Empty Empty) (Node 3 Empty Empty)

Замечания:
- Для того, чтобы проще было тестировать, давайте считать что нумеровать надо в таком порядке: сначала вершина, потом левое поддерево, потом правое поддерево.
- Чтобы тесты прошли, надо обязательно в определении дерева написать deriving Show
